## Remaining Risks

- The runtime feed is still static-file based, so updates require shipping new JSON.
- External embedded cabinets still depend on third-party iframe policy and uptime.
- The flagship game currently uses programmatic canvas visuals instead of sprite art.
- GitHub Pages still must be enabled in repository settings with `GitHub Actions` as the source.
- Older failed workflow runs can still carry duplicate artifacts; they should be replaced by a fresh run, not rerun in place.
